# 🎬 Oracle AI Agent Studio — Workflow Agents Series

A step-by-step video series focused on understanding and building **Workflow Agents** using Oracle AI Agent Studio.

This series is designed for:
- Functional Consultants (Finance, Procurement, HR)
- Technical Developers (APIs, OIC, Integrations)
- Anyone getting started with AI-driven workflows in Oracle Fusion

## 📌 About This Series

This series covers:
- Core concepts of Workflow Agents
- Design patterns and architecture
- Node-level deep dives (AI, Logic, Data)
- Hands-on implementation approaches
- Real-world enterprise use cases

The goal is simple:
> Help you move from understanding → designing → building AI-powered workflows.
