# Sample Invoices

This folder contains sample invoice PDFs used for demonstrating and testing the Oracle Fusion AP Invoice Processor Workflow Agent.

These invoices are used as input documents for the workflow built in Oracle AI Agent Studio.

---

# Purpose

The sample invoices help demonstrate:

* Invoice PDF ingestion
* AI-based invoice extraction
* PO vs Non-PO routing
* Supplier identification
* PO matching
* Invoice payload generation
* Oracle Fusion AP invoice creation

---

# Included Samples

| File                       | Type           | Scenario                          |
| -------------------------- | -------------- | --------------------------------- |
| CDW PO PDF.pdf             | PO Invoice     | Invoice containing Purchase Order |
| Midtown Non PO Invoice.pdf | Non-PO Invoice | Invoice without Purchase Order    |

# Disclaimer

These sample invoices are synthetic demo documents created for testing and demonstration purposes only.
